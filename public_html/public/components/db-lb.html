Laundry Buddy

Preface

Laundry Buddy is the root of Dorm Buddy. As stated in the Overview, Laundry Buddy is the initial idea that led to Dorm Buddy's development. Vincent and I were taking a concurrent programming course at the time, and he wanted to implement this idea of concurrency in the real life laundry machine reservation scenario, and I couldn't agree more

Purpose

Ever go to the laundry room only to find out that all washers are full and you need to go back? Well, this is a common problem on-campus students face, as laundry rooms are shared in each building. Sometimes washers and dryers are full and are never emptied by the students who filled them. This is where Laundry Buddy comes in. Laundry Buddy shows the building's registered laundry machines, and allows students to reserve them ahead of time, state when they started, get notified when the machine is ready, and find out which machines are problematic. This makes the process of doing laundry mindless for students, allowing them to focus on academia instead.

Cycle Templates

When Laundry Buddy is first launched, a screen pops up instructing the user to select washing machine templates in order to view the machines. This is where the user creates their own preferences regarding their cycle. Those preference help remind the user of their settings, and they also determine how long the washing/drying will take, notifying the user when it is done. For washers, a cycle template features the soil level, the type of clothes, the temperature, and the size of the load. For dryers, cycle settings include the temperature and if extra cool down time is needed.

Status and Condition

Before reserving a laundry machine, knowing the status of the machine and its condition is a must. That's why icons were implemented do just that. The icons follow the pictured legend and their explanation is as follows:
  • Status
    • Free - the laundry machine is available for reservation
    • Busy - the machine is currently running a cycle
    • Reserved - the machine has been reserved but no cycle has started
  • Condition
    • Good - this machine is in working order and has no issues
    • Reported - an admnistrator marked this machine as problematic
    • Broken - the machine is out of order. One cannot reserve it

Reservations

If a laundry machine is available, the user can reserve it. Other users will not be allowed to use the machine when it's reserved. After reserving the machine, the user has 15 minutes to go to the laundry room, load the machine, and start their laundry. The reservation gets automatically canceled if they don't start it in time. Once the cycle starts, based on the selected template settings, the machine would run for that period of time. The machine is considered "busy" for the time it takes to complete the cycle, and a 10 minute overhead for users to come pick up their laundry. When the time expires, other users regain the right to reserve the machine.
