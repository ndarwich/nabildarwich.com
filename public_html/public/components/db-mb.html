Map Buddy

Purpose

Map Buddy was designed to aid students in their day to day navigation around campus, and even around the city! A common problem students face in the beginning of a semester, or when they are new to the campus, is that they don't know where different buildings are. That often leads to other problems, such as being late to pivotal first days of classes, not knowing the existence of important buildings, and so on. Map Buddy directly addresses those issues, helping students navigate seamlessly. There are two types of maps in Map Buddy: Interactive user friendly maps, and static local maps.

Interactive Maps

Interactive maps are Map Buddy's flexible user customized maps. Currently, there are two interactive maps: a city map and a campus map. Interactive maps use the Google Maps Android SDK to make map customization possible. When a map is first launched, users get to view and interact with the area, and they are also introduced to a set of predefined markers.

Geolocation Markers

As part of the app, interactive maps already come with static markers situated where hotspots lie. Such hotspots include the recretional center for the campus map, and the nearby shopping mall for the city map. The list of markers can be viewed by tapping the "View Markers" button, and users can tap on any one of the locations to have their map automatically zoom and pan over it. If the user has their location data enabled, they can view where they are relative to the location. If the user feels that a location is not needed anymore, they can long tap it and it disappears after a confirmation. If the user wants to add a new location, they can use the Add Marker interface, where all that is required is giving a name to the marker and tapping its location in the map.

Available Maps

Under available maps, the user gets to see static local maps relevant to where they reside. For now, the available maps include the campus map of the student's college and a map of the user's dormitory floor. The user can select a local map and be able to save it to their phone gallery. When saved, the user can navigate to the Dorm Buddy folder in their gallery and access the local map at their convenience.
